# Data Visualization Project

## Project Overview

This project analyzes e-commerce sales data and creates visualizations to understand sales performance and order status distribution.

## Objectives

- Load and analyze Excel data using Python
- Perform basic data cleaning
- Generate meaningful visualizations
- Extract business insights from the dataset

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- OpenPyXL

## Dataset Information

The dataset contains:

- OrderID
- Date
- CustomerID
- Product
- Quantity
- UnitPrice
- ShippingAddress
- PaymentMethod
- OrderStatus
- TrackingNumber
- ItemsInCart
- CouponCode
- ReferralSource
- TotalPrice

## Data Processing

1. Imported dataset using Pandas
2. Checked data structure and column information
3. Removed missing values
4. Calculated sales values
5. Generated visualizations

## Visualizations

### 1. Product Sales Analysis

A bar chart was created to compare total sales generated by each product category.

### 2. Order Status Distribution

A pie chart was created to visualize the percentage distribution of order statuses.

## Key Insights

- Chair generated the highest sales revenue.
- Printer and Laptop products performed well.
- Phone generated comparatively lower sales.
- Order statuses are distributed almost equally.
- Business can focus on high-performing products to increase revenue.
